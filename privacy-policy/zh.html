隐私政策



简介

本隐私政策旨在告知您关于您提供给我们的任何信息的选择和我们的做法。

使用我们的服务可能涉及在我们的服务（包括我们的网站和移动应用程序）上收集和使用您的信息（如下文"我们收集的信息及收集方式"部分中所定义）。了解这是如何发生的以及您如何控制它非常重要，因此请仔细阅读本隐私政策。

通过使用我们的服务，您同意我们可以按照本隐私政策（不时修订）收集、使用和共享您的信息。

本隐私政策的范围

本隐私政策适用于我们所有的服务。此外，我们的某些服务有适用于其使用的特定隐私指南。此类特定于服务的隐私指南被纳入本隐私政策并构成本隐私政策的一部分。如果任何特定于服务的隐私指南与本隐私政策有所不同，则特定于服务的隐私指南将在差异范围内适用。

本隐私政策被纳入并构成您为使用我们的服务而同意的服务条款的一部分。除非本隐私政策中另有定义或上下文另有要求，否则本隐私政策中使用的任何术语与服务条款中相应的定义术语具有相同含义。

另请注意，本隐私政策不适用于通过以下方式收集的信息：(i) 通过您可能通过我们的服务访问的第三方服务（包括任何第三方网站）；或 (ii) 在我们的服务上宣传其服务的其他公司和组织。

我们如何收集关于您的信息以及为什么？

我们可能直接从您和第三方收集关于您的信息，也可能通过您使用我们的应用程序或服务自动收集。

我们从您和您使用我们的应用程序和服务中收集的信息

我们可能从第三方（例如 Google 或 Apple）接收关于您的某些信息。有关第三方如何收集和披露您的信息，请参阅第三方各自的隐私政策。我们无法控制第三方如何使用或披露您向其提供的个人信息。我们将从第三方接收的信息与我们从您那里收集或收到的关于您的其他信息一起存储。我们接收的信息取决于诸如您正在使用的应用程序或服务、第三方以及您的隐私设置等因素。例如，我们可能收集和存储第三方共享的部分或全部以下信息：
• 您的名字和姓氏；
• 您的个人资料图片或其 URL；
• 您的用户 ID 号码和个人资料信息；
• 您提供给第三方的登录电子邮件；
• 您的性别；
• 您的生日和/或年龄范围；
• 关于您通过第三方进行的活动信息；
• 第三方提供的其他公开可用信息；和/或
• 第三方与我们共享的任何其他信息。

您也可以直接与我们共享信息。应用程序的某些区域和功能可能需要注册、帐户设置和验证，包括验证您已年满 18 岁才能完成注册。您可能被要求向我们提供部分或全部以下信息：

• 您的年龄或生日；
• 您的名字和姓氏；
• 您的电子邮件地址；以及
• 密码。

我们自动收集的信息：设备 ID、Cookie 和其他跟踪技术

我们可能使用 cookie（包括 Flash cookie）、透明 GIF、服务器日志分析和其他跟踪技术（现有的和将来开发的）来增强您使用应用程序和服务的体验，并跟踪有关您使用我们的应用程序和服务的信息。与许多网站一样，我们的网站和代表我们行事的第三方（如 Google Analytics）也存储日志文件并使用 cookie 和其他跟踪技术来收集信息。

例如，我们可能自动收集以下有关您的信息：

• 语言信息、引用页面和退出页面以及 URL；
• 唯一设备标识符（在允许的情况下）；
• 您的计算机或移动设备的操作系统和平台；
• 与您的计算机或移动设备相关联的通信网络，以及移动设备名称和型号；
• 您使用网站、应用程序和服务的日期和时间；以及
• 您使用我们的应用程序的时间长度，和/或访问或使用我们的网站和服务的时间。
我们可能汇总使用此信息，也可能将其与您的个人信息结合使用。
Cookie 是放置在您的设备上的小文本文件，可由放置 cookie 的同一域中的 Web 服务器调用。Cookie 中的文本包含一串数字和字母，可以唯一标识您的设备，也可以包含其他信息。

Web 信标是包含在网站中的电子图像（也称为单像素或透明 GIF）。当您的浏览器打开包含 Web 信标的网页时，它会自动连接到托管该图像的 Web 服务器（通常由第三方托管）。这允许该 Web 服务器记录有关您的设备的信息并设置和读取其自己的 cookie。同样，我们网站上的第三方内容（例如嵌入式视频、插件或广告）会导致您的浏览器连接到托管该内容的第三方 Web 服务器。我们还在我们的促销电子邮件或新闻通讯中包含 Web 信标，以告诉我们您是否打开并对其采取了行动。

移动分析和广告 ID 由移动设备操作系统（iOS 和 Android）生成，应用程序可以访问和使用它们，就像网站访问和使用 cookie 一样。我们的应用程序包含使我们的第三方分析和广告合作伙伴能够访问移动 ID 的软件。

我们和我们的分析和广告合作伙伴在我们的网站、应用程序和在线服务中使用这些技术，在您访问我们的网站（或使用我们的应用程序）时收集个人信息（例如存储在浏览器上的 cookie、移动设备上的广告标识符或设备的 IP 地址）。我们和我们的合作伙伴还使用这些技术来收集有关您随时间跨不同网站或在线服务的在线活动的个人信息。此信息用于存储您的偏好和设置、让您登录、分析我们的网站和应用程序的性能、跟踪您与网站或应用程序的交互、开发推断、提供和定制基于兴趣的广告、打击欺诈以及实现其他合法目的。我们和/或我们的合作伙伴还出于这些目的与第三方共享我们收集或推断的信息。

响应客户的愿望和偏好，包括语言和位置定制、个性化的帮助和说明，或对您和其他客户使用我们的服务的其他响应；

我们的特定于服务的隐私指南可能会更具体地描述我们在相关服务中如何使用个人信息。

如服务条款中的"我们的服务对您的设备的使用"部分所述，为了向您提供服务，我们可能需要访问或使用您设备内的个人信息。我们在您的设备内使用或访问的任何个人信息都将按照本隐私政策和服务条款进行处理。

您的个人信息的共享

除本隐私政策允许或经您同意的情况外，我们不会将您的个人信息转移给任何其他第三方以供其处理用于任何目的。

我们和我们的关联公司可能在我们公司集团内部与合资伙伴以及第三方服务提供商、承包商和代理（例如代表我们发送电子邮件或推送通知的通信服务提供商、协助我们业务的合作伙伴以及我们服务上的广告合作伙伴）共享您的个人信息。此外，我们可能允许此类第三方在我们的服务中收集您的个人信息。在每种情况下，我们允许此类共享和/或收集的目的如下：(i) 向您提供我们的服务；(ii) 协助我们实现上述"我们如何使用您的信息"部分中列出的目的；(iii) 履行我们在服务条款或本隐私政策下的义务并强制执行我们的权利；和/或 (iv) 我们的业务，包括帮助我们更好地理解和改进我们的服务。这些第三方可能位于您的管辖范围之外，并为此目的在这些地方使用和存储您的个人信息。

在我们允许任何第三方按照上述段落收集和使用您的个人信息的情况下，我们将尽合理努力确保此类第三方仅使用您的个人信息：(i) 遵守本隐私政策；以及 (ii) 遵守我们给他们的任何其他指示，包括我们实施的任何适当的保密和安全措施。

您同意第三方使用、共享和转移您的个人信息（在您的管辖范围内外），如本"您的个人信息的共享"部分所述。

随着我们继续发展业务，我们或我们的关联公司可能被第三方收购或重组我们的集团结构。在这种情况下，您的信息可能在我们集团内部转移和/或转移给将继续根据本隐私政策或我们将通知您的不同隐私政策运营我们的服务或类似服务的第三方。您的个人信息转移到的各方可能位于您的管辖范围之外，并在这些地方使用您的信息。

您同意我们或我们的关联公司可能需要保留、保存或披露您的个人信息：(i) 为了遵守适用的法律或法规；(ii) 为了遵守法院命令、传票或其他法律程序；(iii) 响应政府当局、执法机构或类似机构的请求（无论是否位于您的管辖范围内）；或 (iv) 在我们认为合理必要以遵守适用的法律或法规的情况下。您还同意我们或我们的关联公司可能需要披露您的个人信息以强制执行服务条款或本隐私政策，保护我们的权利、财产或安全，或我们关联公司或我们服务其他用户的权利、财产或安全。

关于敏感个人信息的说明

在某些司法管辖区，某些个人信息（例如关于您的种族或民族出身、宗教或哲学观点或个人健康的信息）被归类为"敏感"，并受到比其他个人信息更严格的监管。请注意，您输入到我们服务的内容和信息（例如照片或关于您的学校或社交活动的信息）可能会向其他人披露您的敏感个人信息。我们敦促您考虑在我们的服务中交流敏感个人信息是否合适。

您同意按照本隐私政策所述的目的和方式对敏感个人信息的所有处理。

请注意，我们不使用您的敏感个人信息来向您提供广告。

数据保留和删除

我们通常会在执行我们的服务和遵守我们的法律义务所需的时间内保留您的数据，然后根据我们的内部协议将其删除或擦除。

此外，如果您在很长一段时间内未使用我们的服务，将被视为您不再需要我们的服务，然后我们可能会通过事先通知来擦除您的数据并关闭您的帐户。

但是，需要注意的是，您使用我们的服务进行的通信可能会使您的个人信息落入我们无法控制的第三方手中——例如，如果您之前通过我们的服务公开了您的个人信息。在某些情况下，我们可能无法从我们的服务或系统中删除您的个人信息和/或共享信息——在这种情况下，我们将在合理可行的情况下告知您我们无法这样做的原因。
